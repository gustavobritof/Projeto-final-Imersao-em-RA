Informações do Projeto

Sobre o Jogo

Este é um jogo de cartas em Realidade Aumentada desenvolvido como projeto da imersão em realidade aumentada do IREDE.

A ideia é combinar elementos de estratégia, sorte e tecnologia AR para criar uma experiência única de jogo, fazendo algo similar a jogos de tabuleiro convencionais.

Tecnologias Utilizadas

  • A-Frame: Para exibição 3D e AR
  • MindAR: Para rastreamento dos marcadores
  • Three.js: Biblioteca para gráficos 3D (A-Frame roda em cima do Three.js)
  • HTML5/CSS3/JavaScript: Tecnologias web comuns
